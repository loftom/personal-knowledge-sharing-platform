# 发布说明 v1.0.0

## 版本概述

`v1.0.0` 为个人知识分享与交流平台的首个完整演示版本。
该版本已经具备毕业设计答辩所需的核心业务闭环，覆盖内容发布、审核管理、检索推荐、社区互动、个人空间、分类标签与数据激励等主要模块。

## 本版本已具备的能力

### 1. 内容发布与审核闭环

- 支持文章、教程、问答等内容发布
- 支持内容编辑、草稿保存与提交审核
- 发布与编辑后均需进入审核流程
- 支持敏感词自动拦截
- 支持管理员通过、驳回、下架

### 2. 检索与推荐能力

- 支持关键词检索
- 支持分类、标签、排序筛选
- 支持基础个性化推荐
- 已补充多账号、多标签、多互动演示数据，便于后续推荐测试

### 3. 社区互动能力

- 支持点赞、收藏、评论、回复
- 支持问答、回答与最佳答案
- 支持互动状态回显与持久化展示

### 4. 用户空间与管理能力

- 支持个人主页、我的内容、数据报告、成长中心
- 支持关注与粉丝关系展示
- 支持前后台分离，管理员进入独立后台

## 启动方式

### 后端

```powershell
cd backend
mvn spring-boot:run
```

### 前端

```powershell
cd frontend
npm install
npm run dev
```
