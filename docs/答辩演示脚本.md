# 知识社区平台答辩演示脚本

## 演示准备

- 前台地址：`http://localhost:5173`
- 普通用户登录页：`http://localhost:5173/login`
- 管理员后台入口：普通用户登录后右上角切换管理员账号，再进入 `/admin`

## 测试账号

- 普通用户：`demo_user / demo123456`
- 管理员：`demo_admin / admin123456`
- 访客用户：`demo_viewer / viewer123456`

## 推荐演示顺序

### 1. 前台首页与文章详情

- 打开首页，说明这是普通用户前台。
- 点击任意文章进入详情页。
- 强调页面顶部现在会展示：
  - 标题
  - 作者名
  - 发布时间
  - 阅读、点赞、收藏等数据

### 2. 普通用户发布内容

- 使用 `demo_user` 登录。
- 进入“发布”页面。
- 选择类型、分类、标签。
- 录入标题、摘要和正文。
- 如果想直接展示效果，点击“直接发布”。
- 如果要演示审核流，点击“提交审核”。

### 3. 评论、点赞、收藏

- 切换到 `demo_viewer`。
- 打开刚发布的文章。
- 执行点赞、收藏、发表评论。
- 强调评论区显示的是昵称，不是用户 ID。
- 如果需要，可切回作者账号回复评论，展示二级回复。

### 4. 管理员审核

- 切换到 `demo_admin`。
- 进入 `/admin`。
- 打开“审核台”。
- 展示：
  - 待审核数量
  - 已发布数量
  - 已下架数量
  - 文章待审数
  - 问答待审数
- 选择一篇待审核内容，先点“预览”，再点“通过”。

### 5. 审核后回前台确认

- 切回 `demo_user` 或游客视角。
- 打开首页或搜索页。
- 搜索刚刚审核通过的文章标题。
- 进入详情页，确认文章已经处于可见状态。

### 6. 个人中心与三期能力

- 进入“我的”页面。
- 展示：
  - 我的内容
  - 数据报告
  - 粉丝/关注
  - 成长中心
  - 个人主页
- 在成长中心说明：
  - 积分余额
  - 当前等级
  - 勋章
  - 最近积分流水

## 建议答辩话术

- 一期重点是“内容主链路可运行”，包括注册登录、内容发布、互动、检索和审核。
- 二期重点是“用户体验和社区关系”，包括问答状态流转、最佳答案、通知、粉丝与个人空间。
- 三期重点是“平台化能力”，包括数据报告、积分等级勋章、审核总览与后台管理。

- 如果命令行里中文显示异常，不影响浏览器页面展示。
- 优先以浏览器页面为准进行演示。
- 如果首页未即时刷新，可手动刷新一次页面。
